Linux常用命令合集

发表于2020-08-06|更新于2020-08-06|Linux
|字数总计:4.5k|阅读时长:17分钟|阅读量:|评论数:

ls（查询目录中内容）

格式：ls [选项] [文件或目录]

选项:

-a 显示所有文件，包括隐藏文件（All）

-l 显示详细信息（Long List）ll命令=ls -l

-d 查看目录属性

-h 人性化显示文件大小（Human）

-i 显示inode

文件权限 / 引用计数 / 所有者 / 所属组 /文件大小(字节) /最后一次修改日期 / 文件名称

文件类型：

-：文件 d：目录 l：软链接文件

-rw-r--r--：-rw：所有者 r--：所属组 r--：其他人

文件权限：

r：read，读权限，代表数字4

w：write，写权限，代表数字2

x：excute，执行权限，代表数字1

mkdir（创建目录）

格式：mkdir -p [目录名]

选项:

-p 递归创建（若上级目录不存在，则进行递归创建）

cd（切换所在目录）

格式:

cd ~ 进入当前用户的家目录

cd - 进入上次目录

cd .. 进入上一级目录

cd . 进入当前目录

cd 进入当前用户的家目录

相对路径：参照当前所在目录，进行查找，如:[root@localhost /]# cd ../usr/local/src/

绝对路径：从根目录开始指定，一级一级递归查找。在任何目录下，都能进入指定位置，如：cd /etc/

cat（查看文件内容）

格式：cat [文件名]

选项:

-n 给输出的所有行加上编号

wc（统计文件内容）

格式：wc [文件名]

选项:

-l 统计行数

-w 统计单词数

-c 统计字符数

rmdir（删除空目录）

格式：mkdir [目录名]

rm（删除文件或目录）

格式：rm -rf [文件或目录]

选项:

-r 删除目录

-f 强制（False）

cp（复制命令）

格式：cp [选项] [原文件或目录] [目标目录]

选项:

-r 复制目录

-p 连带文件属性复制

-d 若源文件是链接文件，这复制链接属性

-h 相当于-pdr

mv（剪切或改名）

格式：mv [原文件或目录] [目标目录]

ln（链接命令）

格式：ln [选项] [原文件] [目标文件]

选项:

-s 创建软链接

  • 软链接特征
    • 类似Windows快捷方式
    • 软链接拥有自己的i节点和Block块，但是数据块只保 存原文件的文件名和i节点号，并没有实际的文件数据
    • lrwxrwxrwx l软链接(软链接文件权限都为rwxrwxrwx)
    • 修改任意文件，另一个都改变
    • 删除原文件，软链接不能使用
  • 硬链接特征
    • 用于相同的i节点和存储block块，可以看做是同一个文件
    • 可通过i节点识别
    • 不能跨分区
    • 不能针对目录使用
    • 原文件修改后硬链接文件也会随之修改， 通过硬链接修改文件原文件也会被修改
    • 若删除原文件文件，也依然可以通过硬链接来访问文件

locate（文件搜索命令）

格式：locate 文件名

选项:

-r 复制目录

-p 连带文件属性复制

-d 若源文件是链接文件，这复制链接属性

-h 相当于-pdr

在后台数据库中按文件名搜索，搜索速度更快

/var/lib/mlocate是locate命令搜索的后台数据库

mlocate一天一更新，最新创建的文件无法搜索到，可使用updatedb命令进行强制更新数据库

限制：只能按照文件名进行搜索(无法按照大小，日期）

whereis（命令搜索命令）

格式：whereis 命令名

选项:

-b 只查找可执行文件

-m 只查找帮助文件

which（命令搜索命令）

格式：which 命令名

:号为分隔符，执行命令时会在这些路径中逐个寻找，看该目录下是否有对应的可执行命令，如果自己写的程序需要执行需要写绝对路径，或是放在PATH环境变量下

find（文件搜索命令）

格式：find [搜索范围] [搜索条件]

避免大范围搜索，会非常耗费系统资源

find是在系统当中搜索符合条件的文件名， 如果需要匹配，使用通配符匹配，通配符是完全匹配

Linux中通配符:

*匹配任意内容

？匹配任意字符

[ ] 匹配任意一个中括号内的字符（[cd]匹配的是c或d）

-atime 文件访问时间

-ctime 修改文件属性

-mtime 修改文件内容

-10 10天内修改文件

10 10天当天修改的文件

+10 10天前修改的文件

-25k 小于25KB的文件

25k 等于25KB的文件

+25k 大于25KB的文件

千字节是小写k，兆字节是大写M

选项:

-a and 逻辑与，两个条件都满足

-o or 逻辑或，两个条件满足一个即可

-exec/-ok 命令 {}\;对搜索结果执行操作

grep（搜索字符命令）

格式：grep [选项] 字符串 文件名

选项:

-i 忽略大小写

-v 排除指定的字符串（取反）

find命令和grep命令的区别:

find命令：在系统当中搜索符合条件的文件名，如果需要匹配，使用通配符匹配，通配符是完全匹配 。

grep命令：在文件当中搜索符合条件的字符串，如果需要匹配，使用正则表达式进行匹配， 正则表达式时包含匹配。

man（帮助命令）

格式：man 命令

选项:

-f 相当于whatis 命令

-k 相当于apropos 命令

man的级别（可通过man man命令找到）:

  • 1 ：查看命令的帮助
  • 2 ：查看可被内核调用的函数的帮助
  • 3 ：查看函数和函数库的帮助
  • 4 ：查看特殊文件的帮助(主要是/dev目录下的文件)
  • 5 ：查看配置文件的帮助
  • 6 ：查看游戏的帮助
  • 7 ：查看其它杂项的帮助
  • 8 ：查看系统管理员可用命令的帮助
  • 9 ：查看和内核相关文件的帮助

使用man命令阅读手册页

  • 使用↑↓方向键滚动文本

  • 使用Page Up和Page Down键翻页

  • 按Q或q退出阅读环境

  • 按“/”后查找内容

info（详细目录帮助）

格式：info 命令

Enter：进入子帮助页面（带有*号标记）

u：进入上层页面

n：进入下一个帮助小节

p：进入上一个帮助小节

q： 退出

可以在命令后添加--help选项查询命令选项帮助

chmod（修改权限）

格式：chmod [选项][权限设定字符串] 文件

选项:

-c 若该文件权限确实已经更改，才显示其更改动作

-f 若该文件权限无法被更改也不要显示错误讯息

-v 显示权限变更的详细资料

-R 对目前目录下的所有文件与子目录进行相同的权限变更

权限设定字符串:

格式：[ugoa...][[+-=][rwx]...]

u 表示该文件的拥有者

g 表示与该文件的拥有者属于同一个群体(group)者

o 表示其他以外的人

a 表示这三者皆是

+ 表示增加权限、- 表示取消权限、= 表示唯一设定权限

r 表示可读取，w 表示可写入，x 表示可执行

文件权限： r：read，读权限，代表数字4

​ w：write,写权限，代表数字2

​ x：excute,执行权限，代表数字1

分别表示User、Group、及Other的权限

chown（修改文件拥有）

格式：chown [选项] user[:group] 文件

选项:

-c 显示更改的部分的信息

-f 忽略错误信息

-h 修复符号链接

-v 显示详细的处理信息

-R 处理指定目录以及其子目录下的所有文件

zip（.zip格式压缩）

格式：zip 压缩文件名 源文件 #压缩文件

格式：zip –r 压缩文件名 源文件 #压缩目录

unzip（解压.zip文件）

格式：unzip 压缩文件

gzip(.gz格式压缩)

格式：gzip 源文件 #压缩为.gz格式的压缩文件，源文件会消失

格式：gzip -c 源文件 > 压缩文件 #压缩为.gz格式的压缩文件，源文件会保留

​ 例如：gzip –c canls > canls.gz

格式：gzip -d 压缩文件 #解压缩文件

格式：gunzip 压缩文件 #解压缩文件

格式：gzip -r 目录 #压缩目录下的所有子文件，但是不能压缩目录,目录下的文件全变成.gz格式

bzip2（.bz2格式压缩）

格式：bzip2 源文件 #压缩为.bz2格式，不保留源文件

格式：bzip2 -k 源文件 #压缩之后保留源文件

格式：bzip2 –d 压缩文件 #解压缩,-k保留压缩文件

格式：bunzip2 压缩文件 #解压缩,-k保留压缩文件

tar（打包命令）

格式：tar –cvf 打包文件名 源文件 **

选项:

-c 打包

-x 解打包

-v 显示过程

-f 指定打包后的文件名

-t 查看压缩文件内容

.tar.gz格式压缩（tar.gz格式是先打包为.tar格式，再解压为.gz格式 ）:

.tar.bz2格式压缩:

shutdown（关机与重启）

格式：shutdown [选项] 时间

选项:

-c 取消前一个关机命令

-h 关机

-r 重启

其他关机命令：halt poweroff init 0

其他重启命令：reboot init 6

退出登录命令：logout

系统运行级别:

0 ：关机

1 ：单用户（最小安装，用于系统修复）

2 ：不完全多用户，不含NFS服务（文件共享服务）

3 ：完全多用户（字符界面）

4 ：未分配

5 ：图形界面

6 ：重启

使用cat /etc/inittab设置系统默认启动级别

执行命令：

mount（查询与自动挂载）

格式：mount [-t 文件系统] [-o 特殊选项] 设备文件名 挂载点

若自己添加了光盘挂载，若启动时未放入光 盘，会导致系统启动失败

选项:

-t 文件系统：加入文件系统类型来指定挂载的类型，可以ext3、ext4、iso9660等文件系统(Linux默认是不支持NTFS文件系统)

-o 特殊选项：可以指定挂载的额外选项

警告是挂载是默认挂载读写权限，但是光盘一旦制作完成将无法写入数据，被迫挂为只读选项可以将这条警告当做成功的标志

umount（卸载光盘）

格式：umount 设备文件名或挂载点

当出现mount ：/mnt/cdrom ：target is busy

因为当前处于mnt/cdrom下，所以目标正忙， 退出该目录就可以

w（查看用户登录信息）

格式：userdel [-r] 用户名

命令输出:

USER：登录的用户名

TTY：登录终端

FROM：从那个IP地址登录

LOGIN@：登录时间

IDEL：用户配置时间

JCPU：指的是和该终端连接的所有进程占用的时间。这个时间里并不包括过去的 后台作业时间，但却包括当前正在运行的后台作业所占用的时间。

PCPU ：是指当前进程所占用的时间

WHAT：当前正在运行的命令

useradd（添加用户）

格式：useradd 用户名

选项:

-u 指定UID标记号

-d 指定宿主目录，缺省为/home/用户名

-e 指定账号失效时间

-g 指定用户的基本组名（或UID号）

-G 指定用户的附加组名（或GID号）

-M 不为用户创建并初始化宿主目录

-s 指定用户的登录shell

userdel（删除用户）

格式：userdel [-r] 用户名

添加-r选项时，表示连用户的宿主目录一并删除

last（查询当前登录用户信息）

格式：last

last命令默认是读取/var/log/wtmp文件数据

用户名 / 登录终端 / 登录IP /登录时间 / 退出时间

who（列出所有登陆当前系统中的用户信息）

格式：who

用户名 / 登录终端 / 登录时间 (登录来源IP地址)

lastlog（查询所有用户最后一次登录时间）

格式：lastlog

uname（查看系统内核信息）

格式：uname

选项：

-a 显示主机名、内核版本、硬件平台等详细信息

-r 显示内核版本

last命令默认是读取/var/log/wtmp文件数据

ifconfig（查看IP地址）

格式：ipconfig

名称 网卡类型
eth0 以太网
lo （虚拟）回环设备
ppp0 使用PPP协议的串口设备（通常指调制解调器）
tr0 令牌环（Token Ring）
fddi0 光纤

pwd（查看当前所知目录）

格式：pwd

du（统计目录大小）

选项:

-a 统计时包括所有的文件，而不仅仅只统计目录

-h 以更易读的字节单位（K，M等）显示信息

-s 只统计每个参数所占用空间的总大小

netstat(网络连接情况)

选项:

-a 显示所有活动连接

-n 以数字形式显示

-p 显示进程信息

-t 查看TCP协议相关信息

-u 查看UDP协议相关信息

-r 显示路由表信息

hostname（查看主机名）

永久修改主机名：hostnamectl set-hostname HOSTNAME
